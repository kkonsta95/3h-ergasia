# 3h-ergasia

Σε θεωρητικό πλαίσιο, η ιδέα της Εταιρικής Κοινωνικής Ευθύνης έχει προταθεί από τις ΗΠΑ τη δεκαετία του 90.
Η Εταιρική Κοινωνική Ευθύνη έχει προταθεί από την Ευρωπαική Ένωση ως μέσο για την ανάπτυξη της βιωσιμότητας και την ενσωμάτωση από τις επιχειρήσεις για την περεταίρω ανάπτυξη του ανταγωνιστικού τους πλεονεκτήματος. Η ΕΚΕ λειτουργεί σεβόμενη τα συμφέροντα των ενδιαφερομένων μερών που σχετίζονται με την εταιρεία. Ο στόχος στην ΕΚΕ δεν είναι μόνο η βελτιστοποίηση των κερδών, αλλά και η συμμόφρωση των δραστηριοτήτων τους με τις κοινωνικές και περιβαλλοντικά ορθές πρακτικές.  https://www.researchgate.net/publication/329800254_Corporate_social_responsibility_in_hotel_industry_Empirical_analysis_of_transitional_market
